About Community Risk Assessment

Humanitarian organizations cannot help all out of their scarce resources, when a disaster relief or disaster risk-reduction program is carried out. Therefore they try to find and target those most in need. This beneficiary selection is currently a slow and possibly subjective process, largely because of a lack of data and the tools to interpret them.
This solution aims to fill this information gap by finding and integrating all relevant open or closed data sources on a detailed geographical level, on indicators ranging from hazard exposure to vulnerability to coping capacity. Having these data together and easily accessible saves invaluable time when a disaster strikes. Furthermore, the indicators are combined into composite scores, enabling decision-makers to compare communities, thereby contributing to the process of beneficiary selection.
1. Open Data Collection
For each country, all available risk-related, subnational open data sources are gathered.
We collect data sources relating to humanitarian Risk through Hazards, Vulnerability or Coping Capacity.
We work with subnational data only, preferably on Community level.
Data sources that used to be scattered over the internet is collected and stored by us in one place.
2. Data Integration
By not only storing all data in one database, but also cleaning and transforming all data sources into one format, we provide easy access to complete, timely and qualitative risk-data in times of need.
Moreover, we integrate all indicators through a risk framework (inspired by the INFORM risk framework), which assigns a risk score between 0-10 per community. This enables easy comparison and thus prioritization between areas.
3. Data Visualization and Access
All data and resulting risk scores are visually navigable through this dashboard. Additionally there are export or direct API options for working with the data outside of this dashboard.
4. Prioritization and Data Preparedness
All data in this dashboard can directly be used by decision-makers either before, during or after a disaster. Having the data easily accessible in one place saves invaluable time.
Moreover, the risk scores in this dashboard can be used for prioritizing target areas for a Disaster Risk Reduction project.
The Vulnerability and Coping Capacity components can also - when combined with event-specific impact and hazard data - be used for Impact modeling in the Response or Early Warning Phases.
