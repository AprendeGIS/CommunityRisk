Welcome to the Community Profiles prototype for the Philippines

  • The Community profiling project aims to profile all communities in the world, but pilots with the Philippines.
  • It does so by gathering and integrating all relevant data sources available: open or closed.
  • These data are preferably on the deepest geographical or administrative level possible: the community level.
  • The data include indicators ranging from hazards exposure to socio-ecomomic status to infrastructure.
  • Together these indicators form a resilience or risk profile of a community.
  • The data-preparedness that is created by building up this database saves invaluable time when a disaster strikes.
  • Moreover, the resilience scores make communities comparable to each other, enabling decision-makers to pick priority areas.
